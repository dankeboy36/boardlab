# BoardLab

Vendor-independent board development in VS Code, powered by the Arduino CLI.

BoardLab is not affiliated with Arduino. It uses the official Arduino CLI.

## Status

Early preview. Expect rough edges and breaking changes.

## Features

- Manage boards, ports, and sketches from VS Code.
- Compile, upload, and archive sketches with the Arduino CLI.
- Built-in monitor and plotter views.
- Profiles for repeatable board and toolchain setups.

## Roadmap

- Improve first-time setup and onboarding.
- Expand board discovery, diagnostics, and error messaging.
- Polish monitor/plotter UX and performance.
- Publish public documentation and examples.
